# MINABAZAR

## Instructions

### Change WhatsApp and Call Numbers
To change the WhatsApp and call numbers, locate the `config.js` file in the root directory of the project. Update the values for `whatsappNumber` and `callNumber` as per your requirements.

### Add Products and Images
Products can be added in the `products` folder. Each product should have an image and a corresponding JSON file. To add a new product:
1. Create a new JSON file in the `products` folder with the product details.
2. Upload the corresponding product image in the `images/products` directory, ensuring the names match.

### Customize Brand Colors
You can customize the brand colors in the `styles.css` file located in the `css` directory. Look for the following variables and change their values:
- `--primary-color`: Change this to your brand's primary color.
- `--secondary-color`: Change this to your brand's secondary color.

### Deploy on GitHub Pages
To deploy the MINABAZAR project on GitHub Pages:
1. Push your changes to the `main` branch.
2. Go to the repository settings.
3. Enable GitHub Pages and select the `main` branch as the source.

### Directory Structure Documentation
- `config.js`: Configuration settings for the application.
- `products/`: Folder containing product JSON files.
- `images/`: Folder for product images.
- `css/styles.css`: Stylesheet file for customizing the appearance of the site.
- `index.html`: Main HTML file for the application.
